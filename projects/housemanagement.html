House Management Application



House Management Application

View Code

I set out to write this project shortly after I learned Object Oriented Design with C++. I followed the same methodology propped up by my textbook. I made many improvements to the design and implementation of the program after learning data structures.

I designed this program to allow its user to create a 'House' object, with a variable number of appliances (e.g. doors, lights, air conditioners, etc.) Following creation of a House object, the user can log into their house using its assigned name, by use of Maps. The user can manipulate all appliances, and check on the status of the house. Multiple houses can be created and stored in a map. The program is run in a Console Window.
