This application was developed as an academic project. The assignment asked for a console application that allowed the user to store student and teacher informaton into a database, manipulate their informaton, and tie each person to a bank account.

I wrote the program according to the conditions layed out by my instructor. I stored student and teacher objects into vectors, and used composition to link each student/teacher object to a seperate bank account object.
